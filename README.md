# ion-md-input
A material design inspired text input for Ionic Framework

## Usage
```
<ion-md-input placeholder="Name" type="text"></ion-md-input>
```
You can use any attribute that angular provides.

```
  ng-model=""
  [name=""]
  [required=""]
  [ng-required=""]
  [ng-minlength=""]
  [ng-maxlength=""]
  [ng-pattern=""]
  [ng-change=""]
  [ng-trim=""]
```
